# Feldnotizen — KI und Unterricht

Ein serialisiertes Entwicklungstagebuch über KI im Unterricht – aus der Perspektive eines Lehrers.

Die Feldnotizen erzählen, wie aus einzelnen Experimenten mit künstlicher Intelligenz nach und nach neue Werkzeuge, Unterrichtsideen und Arbeitsweisen entstehen.
